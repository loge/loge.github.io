دوره‌ی رایگان آموزش HTML5

علاقه به ساخت وب‌سایت و یا اپ‌ها دارید؟ خب راه برای یادگیری HTML5 (زبان استاندارد وب) اکنون باز است. دوره‌ای از W3C که توسعه‌دهنده‌ی HTML5 است به صورت رایگان پیش روی شماست.

این دوره‌ی شش هفته‌ای از ماه ژوئن آغاز می‌شود ولی شما می‌توانید که اکنون در edX ثبت‌نام کنید.

پیش‌نیازهای دوره آشنایی با اصول بنیادی HTML و CSS است. (دانستن JavaScript هم ضروری نیست ولی مفید خواهد بود).

البته اگر آشنایی هم با موارد فوق‌الذکر ندارید این یکی دوره‌ی رایگان در خصوص HTML و CSS می‌تواند شما را به‌موقع سر کلاس برساند.

آدرس دوره: https://www.edx.org/course/learn-html5-w3c-w3cx-html5-1x
