کارهایی که بعد از نصب اوبونتو ۱۵.۰۴ باید انجام داد

یک انتشار هیجان‌انگیز دیگر از اوبونتو از راه رسید؛ وروت پُرشور (میمون کوچک شرق و جنوب آفریقا) اسم رمز نسخه ۱۵.۰۴ از سیستم‌عامل محبوب اوبونتو است. چیزهای جدید اضافه شده و در بعضی موارد مثل جایگزینی systemd با upstart تغییراتی را داشته‌ایم. هنوز شاهد یونیتی ۸ نیستیم ولی نسخه‌ی ۷.۳ با سرعت و پاسخ‌دهی بالا حرفی را باقی نگذاشته است. هر بار که نسخه‌ی جدیدی از اوبونتو منتشر می‌شود ما در NoobsLab شیرجه‌ای داخلش می‌زنیم تا ببینیم چه کارهایی می‌تواند تجربه‌ی کاربری را بهبود ببخشد. منظورم کامل نبودن اوبونتو نیست، که البته کامل هست، ولی خب کارهایی هست که بعد از یک نصب تازه می‌تواند قدرت مضاعفی را به سیستم‌عامل ما بدهد. تمام دستکاری‌ها توسط NoobsLab تست شده و شما با هیچ مشکلی مواجه نخواهدی شد. اگر هم مواجه شُدید، ما در خدمتتان هستیم. خب دیگر وقتش رسیده که شروع کنیم…

برای دانلود اوبونتو به این صفحه مراجعه کنید.

۱- فعال کردن گزینه‌ی کمینه کردن توسط یک-کلیک

کمینه کردن (مینیمایز) توسط کلیک در اوبونتو ۱۴.۱۰ و ۱۴.۰۴ وجود داشت ولی به صورت پیش‌فرض غیرفعال شده بود و هنوز در اوبونتو ۱۵.۰۴ غیرفعال است ولی شما می‌توانید آن را از طریق مدیر کامپیز فعال کنید.

برای نصب CCSM دستور زیر را در ترمینال وارد کنید:

sudo apt-get install compizconfig-settings-manager

Dash را با کلیک بر روی آیکن اوبونتو در نوار کناری باز کنید و عبارت CCSM را جستجو کنید. سپس به Unity plugin بروید و تیک Minimize Single Window Applications را بزنید.

۲-غیرفعال کردن جستجوی آنلاین Dash

اگر شما جستجوی برخط در Unity Dash را نمی‌پسندید، به راحتی می‌توانید آن را غیرفعال کنید. به Settings و سپس Privacy بروید و خاموشش کنید.

۳- نصب راه‌اندازها از طریق Additional Drivers

برای تجربه‌ی بهتری کاربری در اوبونتو بهتر است که راه‌اندازهای توصیه شده را نصب کنید. شما می‌توانید بهترین راه‌انداز موجود برای سخت‌افزار خودتان را از طریق Additional Driver انتخاب و نصب یا حذف کنید.

Dash را باز کنید و عبارت Software & Updates یا Additional Drivers را جستجو کنید. بعد از باز کردن ابزار مربوطه به سربرگ Additional Drivers رفته و راه‌اندازهای پیشنهادی را نصب نمائید.

۴- ابزار دستکاری سیستم

Gnome Tweak Tool ابزار قدرتمند و شناخته‌‌شده‌ای است که توسط آن می‌توانید محیط اوبونتو را مدیریت کنید. مثلاً تم، آیکن‌ها، فونت‌ها، مکان‌نما و… را تغییر دهید.

Unity Tweak Tool ابزار پیکربندی برای دسکتاپ یونیتی است که دسترسی به ویژگی‌ها و گزینه‌های پیکربندی را در رابط تر و تمیز و ساده‌ای برای کاربر میسر می‌کند.

برای نصب Tweak Tools دستور زیر را در ترمینال صادر کنید:

sudo apt-get install unity-tweak-tool gnome-tweak-tool

۵- غیرفعال کردن اسکرول‌بار مخفی

حالت اسکرول‌بارها در اوبونتو شاید به دل بعضی‌ها ننشیند.

اگر شما می‌خواهید که آن را به حالتی که در سمت چپ تصویر زیر مشخص است تبدیل کنید دستور زیر را در ترمینال وارد کنید:

gsettings set com.canonical.desktop.interface scrollbar-mode normal

اگر می‌خواهید به حالت قبل بازگردید، این دستور را صادر کنید:

gsettings reset com.canonical.desktop.interface scrollbar-mode overlay-auto

۶- غیرفعال کردن جستجوی بازگشتی در ناتیلوس

جستجوی بازگشتی قابلیتی‌ست که شاید برای همه خوشایند نباشد. فرض کنید شما فایلی را دانلود کرده‌اید که نامش حاوی حروف abc است. در مسیری که فایل را دانلود کرده‌ای انبوهی از فایل‌ها و پوشه‌ها وجود دارد. برای پیدا کردن فایل خود از گزینه‌ی جستجو اقدام می‌کنید و abc را می‌نویسید. اگر در زیرمسیرها (subdirectories) فایل‌ها یا پوشه‌هایی با این ترکیب اسم باشند، در جستجوی شما ظاهر خواهند شد و پیدا کردن فایل مورد نظرتان را دشوار خواهد کرد. از طرفی این قابلیت امکان جستجوی گسترده‌تر در زیرمسیرها را به شما می‌دهد. به هر حال این قابلیت بسته به نظر شما می‌تواند بسیار مفید، و یا شاید آزاردهنده باشد.

این قابلیت با دو دستور ساده، فعال و غیرفعال می‌شود.

برای فعال کردن جستجوی بازگشتی در ناتیلوس دستور زیر را صادر کنید:

gsettings set org.gnome.nautilus.preferences enable-interactive-search false

برای غیرفعال کردن جستجوی بازگشتی و استفاده از حالت Type-ahead در ناتیلوس (تصویر بالا) دستور زیر را صادر کنید:

gsettings set org.gnome.nautilus.preferences enable-interactive-search true

۷- نشان دادن نام کاربری در پنل

به صورت نام پیش‌فرض کاربر وارد شده به سیستم در پنل نمایش داده نمی‌شود. اما شما می‌توانید به راحتی آن را فعال کنید.

دستور زیر را صادر کنید:

gsettings set com.canonical.indicator.session show-real-name-on-panel true

برای حذف نام کاربر از روی پنل هم می‌توانید از دستور زیر کمک بگیرید:

gsettings set com.canonical.indicator.session show-real-name-on-panel false

۸- فعال کردن منوی ترکیبی محلی و تنظیم نوار عنوان + منوها

LIM یا منوی ترکیبی محلی بر روی نوار عنوان هر پنجره‌ی غیر-بیشینه‌ای ظاهر می‌شود. در حالت پیش‌فرض این قابلیت غیرفعال است ولی شما به راحتی می‌توانید آن را فعال کنید. به System Settings > Appearance > Behavior بروید. در Show the menus for a window گزینه‌ی In the window's title bar را انتخاب کنید.

در این حالت منوها به جای نمایش در نوار اصلی، در نوار عنوان خود پنجره ظاهر می‌شوند.

تنظیم نوار عنوان و منو

افزایش و یا کاهش اندازه‌ی منو و نوار عنوان ویژگی تازه‌ای‌ست که به یونیتی اضافه شده. به System Settings > Displays بروید و در Scale for menu and title bars بسته به نیازتان تغییرات را اعمال کنید.

۹- نصب گیمپ، برنامه‌ی کار بر روی تصاویر

گیمپ یک برنامه‌ی آزاد برای ساخت، ویرایش و کار بر روی تصاویر است.

برای نصب آن دستور زیر را در ترمینال وارد کنید:

sudo apt-get install gimp gimp-data gimp-plugin-registry gimp-data-extras

۱۰- دیوار آتش برای اوبونتو

لینوکس نیازی به آنتی‌ویروس ندارد ولی دیوار آتش برای هر سیستم‌عاملی مهم است. UFW به صورت پیش‌فرض نصب شده است ولی این ابزار تحت خط فرمان به صورت غیرفعال است. برای استفاده از رابط گرافیکی می‌توانید GUFW را نصب و از طریق آن با UFW ارتباط برقرار کرده و فعال، و مدیریتش کنید.

برای نصب آن دستور زیر را در ترمینال وارد کنید:

sudo apt-get install gufw

بعد از نصب Dash را باز کنید و عبارت GUFW را جستجو کنید.

۱۱- نصب ادوبی فلش پلاگین

فلش‌پلیر به صورت پیش‌فعال در دسترس نیست ولی شما می‌توانید آن را از طریق مخازن اوبونتو نصب کنید.

تریمنال را باز کنید و دستور زیر را صادر نمائید:

sudo apt-get install flashplugin-installer

به عنوان جایگزین می‌توانید ادوبی فلش را از طریق مخازن شریک اوبونتو نصب کنید. Software-Sources را باز کنید و در سربرگ Other Software تیک Canonical Partners را بزنید.

سپس دستورهای زیر را صادر کنید:

sudo apt-get update &&   sudo apt-get install adobe-flashplugin

همچنین Pipelight (معادل Silverlight) برای مرورگر‌های وب نصب کنید.

۱۲- یکپارچه کردن حساب‌های کاربری آنلاین در اوبونتو

اوبونتو قابلیت‌های مختلفی را برای کار با حساب‌های کاربری آنلاین شامل، فیس‌بوک، گوگل، توییتر، یاهو، هات‌میل و… از طریق دسکتاپ اوبونتو فراهم می‌کند. به Settings > Online Accounts رفته و حساب‌های کاربری خود را پیکربندی کنید.

۱۳- نصب اضافات

اضافات محدود شامل برنامه‌های غیرآزاد (Restricted) در مخازن اوبونتو قرار دارند که در زمان نصب اوبونتو نمی‌توانند نصب شوند ولی شما می‌توانید آن‌ها را نصب نمائید. اضافات محدود شامل قابلیت پخش فایل‌های mp3 و پشتیبانی از فرمت‌های مختلف، فونت‌ها، جاوا، پلاگین فلش، lame و پخش dvd می‌شود. دستور زیر را برای نصب آن صادر کنید:

sudo apt-get install ubuntu-restricted-extras

۱۴- کدک‌ها و فعال کردن پخش DVD

اگر کاربر فایل‌های چند رسانه‌ای هستید و از اوبونتو بدین منظور استفاده می‌کنید، این کدک‌ها به کارتان خواهد آمد. دستور زیر را صادر و آن‌ها را نصب کنید:

sudo apt-get install ffmpeg gstreamer0.10-plugins-ugly gxine libdvdread4 icedax tagtool easytag id3tool lame libxine2-ffmpeg nautilus-script-audio-convert libmad0 mpg321 libavcodec-extra gstreamer1.0-libav

دستور زیر پخش DVD را فعال می‌کند:

sudo /usr/share/doc/libdvdread4/install-css.sh

۱۵- ابزار فشرده‌سازی و باز کردن فایل‌های فشرده

شما می‌توانید فایل‌های 7z, zip, gzip, rar, xz, tar, bz2, xar, tar.gz, tar.bz و… را فشرده و یا باز کنید. برای این کار لازم است که بسته‌های مربوط به آن‌ها را نصب نمائید:

sudo apt-get install p7zip-rar p7zip-full unace unrar zip unzip sharutils rar uudeview mpack arj cabextract file-roller

۱۶- ابزار مدیریت قدرت (Power) لپ‌تاپ‌ها

اگر از لپ‌تاپ استفاده می‌کنید، TLP وlaptop mode tools می‌تواند برای شما بسیار مهم باشد. این ابزارها برای شما امکان ذخیره عمر باتری و یک سری دیگر از ویژگی‌ها را فراهم می‌کند.

laptop mode tools را توسط این دستور نصب نمائید:

sudo apt-get install laptop-mode-tools

۱۷- غیرفعال کردن سیستم گزارش خطا

اگر چیزی کرش کند و با خطا مواجه شود پنجره‌ای باز و اطلاعات مربوط به آن را نمایش می‌دهد. چنانچه شما نمی‌خواهید که برای گزارش آن، اعلان دریافت کنید، به راحتی می‌توانید این اعلان‌ها را خاموش نمائید.

دستور زیر را صادر کنید:

sudo gedit /etc/default/apport

حالا در فایل متنی باز شده عبارت «enabled=1» را در انتها مشاهده خواهید کرد. آن را به «enabled=0» تغییر دهید. سپس تغییرات را ذخیره و خارج شوید. اکنون دستور زیر را صادر کنید:

sudo service apport stop

۱۸- از نو آراستن یونیتی

اگر مشکلی پیش آمد می‌توانید دسکتاپ یونیتی را ریست کنید. کافی‌ست که دستور زیر را صادر نمائید:

sudo apt-get install dconf-tools
dconf reset -f /org/compiz/
setsid unity
unity --reset-icons
